API Reference

Comprehensive API documentation for all FinWorld modules and functions

API Overview

FinWorld provides a comprehensive API for financial AI development. The API is organized into several core modules:

API Organization: The API is organized into logical modules that correspond to the seven-layer architecture.

Data API

The Data API provides interfaces for data acquisition, processing, and management.

Downloaders

Download financial data from various sources.

FMPDownloader

Download data from Financial Modeling Prep API.

Parameters:
  • api_key (str): FMP API key
  • assets_path (str): Path to assets list file
  • data_path (str): Output data path
  • start_date (str): Start date for data download
  • end_date (str): End date for data download
  • level (str): Data frequency (1day, 1min)
from finworld.downloader import FMPDownloader

downloader = FMPDownloader(
    api_key="your_api_key",
    assets_path="configs/_asset_list_/dj30.json",
    data_path="workdir/dj30_fmp_price_1day/price",
    start_date="2020-01-01",
    end_date="2023-12-31",
    level="1day"
)

# Download data
downloader.download()

AlpacaDownloader

Download data from Alpaca Markets API.

Parameters:
  • api_key (str): Alpaca API key
  • secret_key (str): Alpaca secret key
  • base_url (str): Alpaca base URL
  • symbols (list): List of symbols to download
from finworld.downloader import AlpacaDownloader

downloader = AlpacaDownloader(
    api_key="your_api_key",
    secret_key="your_secret_key",
    base_url="https://paper-api.alpaca.markets",
    symbols=["AAPL", "GOOGL", "MSFT"]
)

Processors

Process and transform raw financial data.

Processor

Base processor for data transformation.

Parameters:
  • assets_name (str): Name of asset list
  • data_path (str): Path to input data
  • data_type (str): Type of data (price, feature, news)
  • feature_type (str): Type of features (Alpha158, custom)
from finworld.processor import Processor

processor = Processor(
    assets_name="dj30",
    data_path="workdir/dj30_fmp_price_1day/price",
    data_type="price",
    feature_type="Alpha158"
)

# Process data
processor.process()

Datasets

Dataset classes for different financial tasks.

SingleAssetDataset

Dataset for single asset trading tasks.

Parameters:
  • symbol (str): Asset symbol
  • level (str): Data frequency
  • start_timestamp (str): Start timestamp
  • end_timestamp (str): End timestamp
  • num_features (int): Number of features
  • if_norm (bool): Whether to normalize data
from finworld.data import SingleAssetDataset

dataset = SingleAssetDataset(
    symbol="AAPL",
    level="1day",
    start_timestamp="2020-01-01",
    end_timestamp="2023-12-31",
    num_features=154,
    if_norm=True
)

# Get data
data = dataset.get_data()

MultiAssetDataset

Dataset for multi-asset portfolio tasks.

Parameters:
  • symbols (list): List of asset symbols
  • level (str): Data frequency
  • start_timestamp (str): Start timestamp
  • end_timestamp (str): End timestamp
  • num_features (int): Number of features per asset
from finworld.data import MultiAssetDataset

dataset = MultiAssetDataset(
    symbols=["AAPL", "GOOGL", "MSFT", "AMZN"],
    level="1day",
    start_timestamp="2020-01-01",
    end_timestamp="2023-12-31",
    num_features=154
)

Model API

The Model API provides interfaces for various AI models and architectures.

Base Model

Base class for all models in FinWorld.

Model

Base class for all models in the FinWorld library.

from finworld.models.base import Model

class MyCustomModel(Model):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super().__init__()
        self.encoder = nn.Linear(input_dim, hidden_dim)
        self.decoder = nn.Linear(hidden_dim, output_dim)
    
    def forward(self, x):
        x = self.encoder(x)
        return self.decoder(x)

Time Series Models

Models for time series forecasting tasks.

Autoformer

Autoformer model for time series forecasting.

Parameters:
  • input_dim (int): Input dimension
  • output_dim (int): Output dimension
  • seq_len (int): Input sequence length
  • pred_len (int): Prediction length
  • d_model (int): Model dimension
  • n_heads (int): Number of attention heads
from finworld.models.time import Autoformer

model = Autoformer(
    input_dim=154,
    output_dim=1,
    seq_len=96,
    pred_len=24,
    d_model=512,
    n_heads=8
)

Crossformer

Crossformer model for time series forecasting.

from finworld.models.time import Crossformer

model = Crossformer(
    input_dim=154,
    output_dim=1,
    seq_len=96,
    pred_len=24,
    d_model=512,
    n_heads=8
)

RL Models

Models for reinforcement learning tasks.

PPO

Proximal Policy Optimization model.

Parameters:
  • actor (dict): Actor network configuration
  • critic (dict): Critic network configuration
  • gamma (float): Discount factor
  • clip_ratio (float): PPO clip ratio
from finworld.models.rl import PPO

model = PPO(
    actor=dict(
        type="Actor",
        state_dim=64,
        action_dim=3,
        hidden_dim=128
    ),
    critic=dict(
        type="Critic",
        state_dim=64,
        hidden_dim=128
    ),
    gamma=0.99,
    clip_ratio=0.2
)

SAC

Soft Actor-Critic model.

from finworld.models.rl import SAC

model = SAC(
    actor=dict(
        type="Actor",
        state_dim=64,
        action_dim=3,
        hidden_dim=128
    ),
    critic=dict(
        type="Critic",
        state_dim=64,
        action_dim=3,
        hidden_dim=128
    ),
    temperature=0.1
)

Training API

The Training API provides interfaces for model training and optimization.

Trainers

Training classes for different tasks.

TradingTrainer

Trainer for trading tasks.

Parameters:
  • model (Model): Model to train
  • dataset (Dataset): Training dataset
  • environment (Environment): Training environment
  • optimizer (dict): Optimizer configuration
  • scheduler (dict): Scheduler configuration
from finworld.trainer import TradingTrainer

trainer = TradingTrainer(
    model=model,
    dataset=dataset,
    environment=environment,
    optimizer=dict(
        type="AdamW",
        lr=3e-4,
        weight_decay=0.01
    ),
    scheduler=dict(
        type="CosineAnnealingLR",
        T_max=1000
    )
)

# Train model
trainer.train()

PortfolioTrainer

Trainer for portfolio management tasks.

from finworld.trainer import PortfolioTrainer

trainer = PortfolioTrainer(
    model=model,
    dataset=dataset,
    environment=environment,
    optimizer=dict(
        type="AdamW",
        lr=3e-4
    )
)

Evaluation API

The Evaluation API provides interfaces for model evaluation and metrics.

Metrics

Financial-specific metrics for model evaluation.

ARR (Annualized Rate of Return)

Calculate annualized rate of return.

from finworld.metric import ARR

metric = ARR(level="1day")
result = metric.calculate(returns, benchmark_returns)

SR (Sharpe Ratio)

Calculate Sharpe ratio.

from finworld.metric import SR

metric = SR(level="1day")
result = metric.calculate(returns, benchmark_returns)

MDD (Maximum Drawdown)

Calculate maximum drawdown.

from finworld.metric import MDD

metric = MDD(level="1day")
result = metric.calculate(returns)

Evaluators

Comprehensive evaluation classes.

TradingEvaluator

Evaluator for trading tasks.

from finworld.evaluator import TradingEvaluator

evaluator = TradingEvaluator(
    metrics=[ARR(), SR(), MDD()],
    level="1day"
)

results = evaluator.evaluate(
    returns=portfolio_returns,
    benchmark_returns=benchmark_returns
)

Agent API

The Agent API provides interfaces for AI agents and multi-agent systems.

Agents

AI agents for financial tasks.

FinanceAgent

Financial reasoning agent.

Parameters:
  • model (str): LLM model name
  • name (str): Agent name
  • description (str): Agent description
  • tools (list): List of available tools
  • max_steps (int): Maximum reasoning steps
from finworld.agent import FinanceAgent

agent = FinanceAgent(
    model="Qwen3-32B",
    name="finance_agent",
    description="Finance agent for trading decisions",
    tools=[],
    max_steps=20
)

# Run agent
result = agent.run(task_description, context)

GeneralAgent

General-purpose agent with tool access.

from finworld.agent import GeneralAgent

agent = GeneralAgent(
    model="gpt-4.1",
    name="general_agent",
    description="General agent with tool access",
    tools=[
        dict(type="python_interpreter_tool"),
        dict(type="web_fetcher_tool")
    ],
    max_steps=20
)
